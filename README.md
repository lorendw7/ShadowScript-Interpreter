# 影语解释器（ShadowScript Interpreter）
## 一、项目简介
影语解释器是一款基于 Python 实现的轻量级自定义脚本解释器。

## 二、核心特性
### 1. 轻量化易上手
- 核心代码模块化拆分（词法分析、语法分析、解释执行分离），无复杂第三方依赖，零基础也可快速搭建运行环境；
- 代码注释清晰，关键逻辑（如 Token 解析、AST 构建）均有明确说明。

### 2. 完整的基础解释器能力
- **词法分析**：支持将脚本源代码拆分为关键字、运算符、标识符、字面量等核心 Token，自动跳过注释与空白符；
- **语法分析**：基于递归下降解析法构建抽象语法树（AST），支持变量声明/赋值、算术表达式、条件判断、循环结构等基础语法；
- **解释执行**：实现 AST 节点遍历与运行，维护全局/局部作用域，支持流程控制，可交互式运行。

## 三、核心功能清单
| 功能模块       | 具体能力                                                                 |
|----------------|--------------------------------------------------------------------------|
| 词法分析（Lexer） | 关键字/运算符/标识符解析、注释/空白符过滤、基础语法符号校验               |
| 语法分析（Parser） | 抽象语法树（AST）构建、支持变量/表达式/条件/循环的语法解析           |
| 解释执行（Interpreter） | 作用域管理、表达式求值、流程控制执行               |
| 辅助能力       | REPL 交互式运行、脚本文件加载执行、基础错误捕获与提示                     |
